## CommonJS

`
//test.js
var name = '1';

// index.js
require('./test.js');
` 
test.js会形成一个属于模块自身的作用域，所有变量及函数外部不可见。

exports.name = 2;
module.exports = {
    name: 1
}

export.name = 2;相当于在module.exports对象中加个name属性。


require('./test.js');只会执行一遍。有个属性loaded用于记录该模块是否被加载过。默认是false，第一次被加载后变为true，后续检测到就不会加载了。
